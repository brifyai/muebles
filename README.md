# ErgoCraft - Modern Furniture Store

This is a comprehensive clone of the ErgoCraft website, built with React, Vite, and Tailwind CSS.

## Features

- **Responsive Design**: Fully responsive layout for desktop and mobile.
- **Modern UI**: Clean and modern interface with smooth animations and custom typography.
- **Components**:
  - **Navbar**: Sticky navigation with mobile menu.
  - **Hero Section**: Animated split layout with call-to-action.
  - **Logo Strip**: Trusted partner logos.
  - **Featured Categories**: Image grid for product categories.
  - **Best Sellers Product Grid**: interactive product cards.
  - **Promo Banner**: Full-width promotional section.
  - **Features Section**: Highlight of key benefits (Shipping, Warranty, etc.).
  - **Testimonials**: Customer reviews and social proof.
  - **Blog Preview**: Latest articles section.
  - **Newsletter**: Subscription CTA.
  - **Footer**: Comprehensive footer with links.

## Technologies Used

- React 19
- Vite
- Tailwind CSS v4
- Framer Motion (Animations)
- Lucide React (Icons)
- Google Fonts (Outfit & Inter)

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
